📖 关于 SSP Tooling

SSP (Sakura Script Player) 是一款功能强大的伺か兼容桌面角色软件。本工具包提供：

  • SSP 本体一键安装（自动下载 Java 运行环境）
  • 简体中文语言包集成
  • 繁体中文语言包集成
  • 自动更新与配置工具

本项目基于 ukatech/ssp-sinicization 的汉化资源构建。

什么是伺か (Ukagaka)?

伺かは、デスクトップ上で動作するキャラクター表示エンジンです。ゴーストと呼ばれるキャラクターが話しかけたり、デスクトップを彩ります。

Ukagaka is a desktop character engine. "Ghosts" (characters) live on your desktop, talk to you, and provide various functions.

🔧 使用方法 / Usage

  1. 下载 ssp_2_8_31f.exe
  2. 双击运行（会自动下载 Java 17 运行环境，仅首次需要）
  3. 安装程序会自动部署 SSP 和语言包
  4. 启动 SSP，享受中文界面！
