  • 左側の「登録」から新規課題を登録できます。

  • 不具合報告前にOSを「再起動」してみてください。OSレベルでの問題が直る場合があります。
    特に最近のWindowsの場合 シャットダウン→電源ON と 再起動 で挙動が異なります。
    Please reboot OS before reporting problem. It resolves OS level issue.
    In particular, recent Windows behave differently between shutdown and restart.
